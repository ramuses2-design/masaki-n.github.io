用語・索引

用語メモ

  • QA（品質保証） … 出荷後も含めてお客様に品質を“約束”する広い活動。
  • QC（品質管理） … 主に工程内で基準を満たすかを管理する活動。
  • 属人化 … 特定業務が特定個人に依存している状態。副担当で解消する。
  • 特採（特別採用） … 規格外でも条件付きで採用すること。出荷可否は要・上位判断。
  • 目付（めつけ／坪量） … 単位面積あたりの質量（g/m²）。不織布の基本特性で、多くの不良と相関する。
  • 地合い（じあい） … 繊維分布の均一さ。ムラ・透け・強度ばらつきの元になる。
  • 通気度 … 空気の通りやすさ。フィルター・衛生材など用途で重要な機能特性。
  • MD／CD … 流れ方向（Machine Direction）と幅方向（Cross Direction）。強度・伸度は方向で差が出る。
  • 4M … Man（人）・Machine（機械）・Material（材料）・Method（方法）。変化点管理の基本軸。
  • 8D／CAPA … 是正処置の型。8D＝8ステップ、CAPA＝是正・予防処置。B4で使う。
  • MSA … 測定システム解析。測定のばらつき（Gage R&R 等）を評価する手法。
  • chemSHERPA … 製品含有化学物質の情報を伝達する共通スキーム。IMDSと並ぶ代表的な仕組み。
  • SPC（統計的工程管理） … 管理図などでばらつきを監視し、異常を早期に検知する手法。
  • Cp／Cpk（工程能力指数） … 規格に対する工程のばらつき余裕度。Cpkは中心ずれも考慮する。
  • ポカヨケ … 間違いを物理的・仕組み的に防ぐ工夫。取り違え・付け忘れの防止に使う。
  • スパンボンド／メルトブロー … 代表的な不織布の製法。長繊維系と極細繊維系で特性が異なる。
  • なぜなぜ分析（5 Whys） … 「なぜ」を繰り返し真因に迫る手法。B3/B4の原因究明で使う。
  • 限度見本 … 合否の境目を示す実物見本。目視検査の基準を揃えるために用いる。
  • ΔE（色差） … 2色の差を数値化した指標。色調・白度の合否管理（TS18）で使う。
  • PFMEA（工程FMEA） … 工程の故障モードと影響を事前に洗い出し、優先度をつけて対策する手法（T11）。
  • 層別監査（LPA） … 役職を横断した層別の定期工程監査。標準・現物・記録を現場で確認する（D8）。
  • FIFO（先入れ先出し） … 先に入れた在庫から先に使う運用。滞留・期限切れの防止に有効（P9）。
  • コントロールプラン … 工程ごとの管理項目・基準・方法・頻度をまとめた管理計画表。
  • 圧力損失（圧損） … 通気時の圧力の落ち込み。通気度・ろ過性能の指標（TS19）。
  • 剛軟度（剛軟性） … 布・不織布の曲げやすさ（硬さ・柔らかさ）の指標。風合い・ドレープ性の評価に使う（TS21）。
  • スリット／断裁 … 広幅の原反を製品幅・長さに切り分ける工程。切断面の毛羽・バリ・寸法品質に影響（TS20）。
  • SPC管理図（Xbar-R 等） … 工程のばらつきを時系列で監視し、異常を早期検知する管理図（T12）。
  • リサイクル素材／GRS … 再生原料の使用。GRS等の認証は再生材含有率とトレーサビリティを求める（S12）。
  • 選別（ソーティング） … 良品・不良品を基準で仕分ける作業。流出後の切り分け・回収判断で行う（B9）。
  • エンボス／型付け … ロールの凹凸で不織布に模様・立体を付与する加工。欠け・つぶれ・位置ズレが外観/風合いに影響（TS22）。
  • 紙管／コア … 巻取りの芯となる管。強度不足・変形はコアマークや巻き崩れの原因（TS23）。
  • トレーサビリティ … 原料〜工程〜出荷をロット・識別で追える状態。流出時の範囲特定に不可欠（T13・B9）。
  • チョコ停 … 短時間の設備停止。頻発すると品質異常・生産性低下の兆候になる（M13）。
  • 通商措置／関税 … 輸入品にかかる追加関税等の政策。調達コスト・供給に影響し、調達リスク管理の対象（P11）。
  • ラミネート／貼合 … フィルム等と貼り合わせて複合材にする加工。層間剥離・気泡・シワが品質に影響（TS24）。
  • 検針／金属検出 … 折れ針・金属異物を検出する工程。テストピース照合で感度を担保する（TS25）。
  • SCAR（是正処置報告書） … 顧客へ提出する不適合の是正報告様式。8Dと同様の型で作成する（B8）。
  • 受入不良率（PPM） … 受入100万個あたりの不良数。サプライヤー品質のスコアリングに使う（P12）。
  • 相見積り（あいみつ） … 複数社から見積りを取り比較すること。価格以外の品質価値の訴求が重要（S14）。
  • 第一種特定化学物質 … 化審法で難分解・高蓄積・長期毒性等が認められ、製造・輸入・使用が原則禁止される物質。新規指定時は含有製品の輸入も規制対象になり得る（TS26）。
  • 長鎖PFCA（LC-PFCA） … 炭素鎖の長いペルフルオロカルボン酸類。撥水・防汚等に関連。規制強化が進む有機フッ素化合物（PFAS）の一群（TS26）。
  • MCCP（中鎖塩素化パラフィン） … 潤滑・可塑・難燃等に用いられてきた塩素化パラフィン。残留性等の懸念から規制対象化が進む（TS26）。
  • 化学物質管理者 … 安衛法でリスクアセスメント対象物質を扱う事業場に選任が義務づけられた管理担当。ラベル・SDS・リスクアセスメントの統括を担う（TS26）。
  • リスクアセスメント対象物質 … 安衛法でSDS交付・リスクアセスメントが義務づけられた化学物質。対象は段階的に拡大しており、該非確認の定期更新が必要（TS26）。
  • ノーコード／ローコード … プログラミングをほとんど書かずに業務アプリやワークフローを作れる仕組み。現場・管理部門が自分たちで内製できる反面、管理責任者と記録要件を先に決めておく必要がある（T15）。
  • 承認ワークフロー … 申請→承認→記録の流れを電子化した仕組み。承認者・期限・リマインドを設計しないと「押すだけ」の形骸化に陥る（T15・TS27）。
  • 監査証跡（オーディットトレイル） … 記録を「誰が・いつ・何を変えたか」を追える履歴。電子記録の信頼性を担保する要（D5・TS27）。
  • 正本（せいほん） … 同じ記録が複数ある場合に「これが正式」と定めた版。紙と電子の並行運用時は必ず1つに決める（D5）。
  • 濃度基準値 … 安衛法で定められた化学物質のばく露濃度の上限基準。基準値が定められた物質は、測定によりばく露が基準以下であることの確認が求められる（D4）。
  • 見える化 … 現状を誰でも同じように把握できる状態にすること。データを表示するだけでは不十分で、打ち手につながって初めて成立する（D6・TS28）。
  • 予兆保全（予知保全） … 温度・振動・電流などの傾向から故障の兆しをつかみ、壊れる前に手を打つ保全。事後保全・時間基準保全の次の段階（T9・TS28）。
  • アラート疲れ … 通知が多すぎる・当たらない状態が続き、現場が警告を無視するようになること。閾値の見直しとレベル分けで防ぐ（TS28）。
  • OEE（設備総合効率） … 時間稼働率×性能稼働率×良品率で設備の実力を表す指標。品質と生産性を1つの数字で結ぶ（D6・T9）。
  • デジタルツイン … 現実の設備・工程を仮想空間に再現し、シミュレーションで検討する仕組み。見える化・分析・自動化の先にある段階で、いきなり目指すものではない（TS28）。
  • バイオベース／生分解性素材 … 植物由来原料を使う、または微生物等で分解される素材。環境対応で採用が進む一方、既存材より物性のばらつきが大きく、切替時は品質妥当性の再確認が要る（TS29・S12）。
  • 再生材（リサイクル材） … 使用済み・端材を再利用した原料。含有率・トレーサビリティの裏付けとロット間ばらつきの監視が必要（TS29・P8）。
  • СoA（試験成績書／Certificate of Analysis） … 供給側が発行する原材料の試験結果証明。グレード規格や物性差の確認に使う（TS29・P2）。
  • ハルシネーション … 生成AIがもっともらしい誤りを出力する現象。Copilotの出力も必ず人が事実確認する前提で扱う（T16）。
  • GビズIDプライム … 補助金の電子申請等に必要な事業者向け共通認証アカウント。取得に日数がかかるため、補助金検討段階で早めに準備する。
  • トリアージ … 異常の第一報を受けた時点で、原因究明より先に「どこまで急ぐか」を上から順に決めること。安全→法規→顧客影響→波及範囲→社内損失の順で確認し、迷ったら上位（安全側）に倒す（B1）。
  • ラストバイ（最終手配） … 材料・部品の製造中止（EOL）予告を受けて、切替が済むまでの必要数をまとめて最終購入すること。数量の読み違いが欠品・過剰在庫に直結するため、切替計画とセットで決める（C5・P3）。
  • 初物確認（初品確認） … 品種切替・4M変更・段取り替えの直後の最初の製品を、流す前に確認して合否を決めること。変化点の直後は不良が出やすく、ここで止めれば流出しない（M7・C1・C5）。
