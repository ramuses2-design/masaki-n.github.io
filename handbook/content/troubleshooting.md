# トラブルシューティング（症状から引く）

> **位置づけ**：不織布製造で起こりやすい不良を「症状」から引けるようにした**たたき台**。
> 各症状ごとに「まず確認（初動）→ 起こりやすい要因（4M）→ 切り分け → 恒久対策の方向」を統一の型で整理する。
> **使い方**：異常時はまず該当症状を開き、初動チェックで**止血**してから、4Mで切り分ける。原因究明・是正の本手順は場面 B（異常対応）を参照。
> **凡例**：`★`＝外すと事故・流出につながる重点。`【要確認】`＝規格値・設備名・判定基準など、当社の実データで確定が必要な箇所（数値は現時点で埋めていない）。
> **共通の型**：① 現品確保・記録 → ② 止める・隔離 → ③ 4Mで切り分け → ④ 恒久対策・横展開（B2〜B4と同じ流れ）。

## TS1. 異物・黒点（コンタミネーション）
**症状**：製品中に黒点・異物・毛髪・虫・樹脂の焦げ・金属片などが混入。お客様工程やクリーン用途では特に重大。

**まず確認（初動）**
- [ ] 現品・異物を確保し、発生ロット・位置・頻度を記録／写真 ★
- [ ] 出荷停止・該当ロット隔離、前後ロットへの波及確認 ★
- [ ] 異物の種類を特定（金属／樹脂／繊維／虫／油）→ 発生源の見当をつける

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 人 | 手袋・帽子・入退室ルールの不徹底、私物持込 |
| 機械 | 設備摩耗粉・削れ、潤滑油の飛散、フィルタ劣化 |
| 材料 | 原料への異物混入、再生材・回収品の管理不足 |
| 方法/環境 | 清掃不足、5S不良、異物侵入経路（開口部）、空調・粉塵 |

**切り分け・確認**
- [ ] 異物が「金属」なら金属探知機・マグネットの有無と設定を確認
- [ ] 特定工程・特定号機に偏っていないか（層別）
- [ ] 材料由来か工程由来かを、原料検査・工程内サンプルで切り分け ★

**恒久対策の方向**
- [ ] 発生源対策（設備カバー・フィルタ・清掃基準）＋流出対策（検査・探知機）の両面 ★
- [ ] 5S・異物混入防止ルールの再教育、清掃点検の標準化
- [ ] 【要確認】異物の社内許容基準（サイズ・個数）・検出手段

関連場面: B2, B3, D4

## TS2. 目付（坪量）ばらつき
**症状**：目付（単位面積重量）が規格上下限を外れる／ロット内・幅方向でばらつく。

**まず確認（初動）**
- [ ] 測定方法・サンプリング位置が基準どおりか確認（測定起因を除外）★
- [ ] 該当ロットの目付データの傾向（幅方向・流れ方向）を確認
- [ ] 規格外ロットを隔離、出荷可否を保留

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 機械 | 供給量・ライン速度の変動、計量・フィード系の不調 |
| 材料 | 原料の嵩・水分・ロット差 |
| 方法 | 条件設定ミス、幅方向プロファイルの未調整 |
| 測定 | 秤・サンプル面積・採取位置の誤差 |

**切り分け・確認**
- [ ] 秤・サンプリングを再確認し、測定ばらつきを排除（MSA的視点）★
- [ ] 幅方向プロファイルを測定（片側偏り＝供給/エア分布）
- [ ] 時系列で変動か、突発か（変化点の有無）

**恒久対策の方向**
- [ ] 供給量・速度の管理幅を設定し、工程能力（Cp/Cpk）で監視
- [ ] 幅方向プロファイルの定期確認・自動制御の検討
- [ ] 【要確認】目付の規格・管理幅・測定手順（サンプル面積・点数）

関連場面: D1, D6, C1

## TS3. 地合い不良（ムラ・不均一）
**症状**：繊維分布が不均一で、地合い（見た目の均一さ）が悪い。斑・雲状ムラ・筋状ムラ。

**まず確認（初動）**
- [ ] 限度見本と照合し、合否を判定（官能のブレを防ぐ）★
- [ ] 発生範囲（全幅／部分、連続／断続）を記録

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 機械 | 開繊・カード・エア分散の不良、ノズル詰まり |
| 材料 | 繊維の開繊性・長さ・捲縮のばらつき、静電気 |
| 方法/環境 | 条件（速度・エア圧）不適、温湿度による静電 |

**切り分け・確認**
- [ ] ムラのパターンで発生系統を推定（周期性＝機械、ランダム＝材料/分散）
- [ ] 開繊・分散工程の点検（詰まり・摩耗）
- [ ] 湿度・静電の管理状態を確認

**恒久対策の方向**
- [ ] 開繊・分散条件の最適化、ノズル・部品の清掃/交換基準化
- [ ] 静電・湿度管理、材料受入での開繊性確認
- [ ] 【要確認】地合いの判定方法（限度見本・画像評価）と合否基準

関連場面: D1, C4, TS2

## TS4. 幅・寸法不良（幅／厚み／長さ）
**症状**：製品幅・厚み・巻き長が規格外。スリット幅ばらつき、耳（端部）の寸法不良。

**まず確認（初動）**
- [ ] 測定器・測定位置・タイミングが基準どおりか確認 ★
- [ ] 規格外ロットの隔離、範囲（連続/突発）の特定

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 機械 | スリッター刃・ガイド・張力（テンション）の不良 |
| 材料 | 熱収縮・伸びのロット差 |
| 方法 | 条件（温度・張力・速度）設定、位置決め |
| 測定 | 測定器の校正・読み取り |

**切り分け・確認**
- [ ] 張力・温度による収縮/伸びの影響を確認
- [ ] スリッター刃・ガイドの摩耗/位置ずれ点検
- [ ] 測定器の校正状態を確認（期限・ゼロ点）★

**恒久対策の方向**
- [ ] 張力・温度の管理幅設定と監視、刃・ガイドの交換基準
- [ ] 収縮特性を見込んだ条件設定
- [ ] 【要確認】幅・厚み・長さの規格値と測定手順

関連場面: D2, D1, C1

## TS5. キズ・スジ・線キズ
**症状**：流れ方向の連続スジ、押し当てキズ、擦れ跡。

**まず確認（初動）**
- [ ] 発生位置（幅方向の同じ位置か）を記録＝接触部品の特定に直結 ★
- [ ] 該当ロット隔離、いつから発生かを特定

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 機械 | ロール・ガイド・バーの傷/異物付着、回転不良 |
| 材料 | 硬い異物の噛み込み |
| 方法 | 張力過大、接触圧の過大 |

**切り分け・確認**
- [ ] 幅方向で位置が一定＝固定接触部品、移動＝随伴異物
- [ ] 搬送経路の接触部品を順に点検（傷・付着物・回転）★

**恒久対策の方向**
- [ ] 該当部品の研磨/交換・清掃、非接触化・張力見直し
- [ ] 定期点検項目に接触部品の傷確認を追加
- [ ] 【要確認】キズの許容基準（長さ・深さ・本数）

関連場面: B3, D1, TS1

## TS6. 毛羽・毛玉・繊維脱落（発塵）
**症状**：表面の毛羽立ち、毛玉、繊維の脱落・発塵。清浄用途・医療用途で重大。

**まず確認（初動）**
- [ ] 発塵/脱落の程度を限度見本・試験で確認 ★
- [ ] 用途（クリーン/医療）に照らし出荷可否を保留

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 機械 | 接着/融着不足、カッター切れ味、搬送の擦れ |
| 材料 | 繊維長・強度・接着性のばらつき |
| 方法 | 接着条件（温度・圧・時間）不足 |

**切り分け・確認**
- [ ] 接着・融着強度を確認（TS8と連動）
- [ ] カッター・スリッター刃の切れ味点検
- [ ] 材料の接着性・繊維特性を確認

**恒久対策の方向**
- [ ] 接着条件の最適化、刃の交換基準、搬送の擦れ低減
- [ ] 発塵試験の導入（清浄用途）
- [ ] 【要確認】毛羽・発塵の評価方法と合否基準

関連場面: TS8, D1, C4

## TS7. シワ・折れ・カール
**症状**：流れ方向/幅方向のシワ、折れ込み、端部のカール（巻き上がり）。

**まず確認（初動）**
- [ ] 発生パターン（斜め/直線/端部）と範囲を記録 ★
- [ ] 該当ロット隔離

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 機械 | ロールの平行度/アライメント不良、テンション不均一、拡幅不足 |
| 材料 | 熱収縮差、片伸び |
| 方法/環境 | 温度・張力バランス、冷却不足、湿度 |

**切り分け・確認**
- [ ] テンション・アライメント・拡幅装置を点検
- [ ] 冷却・温度プロファイルを確認（カールは収縮差が主因）

**恒久対策の方向**
- [ ] テンション/アライメント調整、拡幅・冷却条件の最適化
- [ ] 【要確認】シワ・カールの許容基準

関連場面: TS4, C1, D1

## TS8. 接着・熱融着不良（強度・剥離）
**症状**：接着/熱融着が弱く剥がれる、逆に硬すぎる・溶けすぎ。層間剥離、エンボス不良。

**まず確認（初動）**
- [ ] 接着/融着強度・剥離を試験で確認、規格と照合 ★
- [ ] 規格外ロット隔離、範囲特定

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 機械 | 温度・圧力・エンボスロールの不良、ヒータムラ |
| 材料 | 樹脂/バインダのロット差、融点ばらつき |
| 方法 | 温度・圧力・速度・時間の条件不適 |
| 測定 | 強度試験の条件・治具 |

**切り分け・確認**
- [ ] 温度・圧力・速度の実測と設定の乖離を確認 ★
- [ ] 幅方向・号機での差（ヒータ/ロールムラ）
- [ ] 材料ロット差の切り分け（受入・成績書）

**恒久対策の方向**
- [ ] 接着条件の管理幅設定と監視、ロール/ヒータの保全
- [ ] 材料の受入基準（融点・目付）強化
- [ ] 【要確認】接着強度の規格・試験方法（速度・治具）

関連場面: B3, D6, C1, TS22

## TS9. 巻き不良（巻きズレ・巻き硬さ・耳不良）
**症状**：巻きズレ（テレスコープ）、巻き硬さの過不足、耳（端部）の不揃い、巻きシワ。

**まず確認（初動）**
- [ ] 巻き状態を記録（ズレ量・硬さ・耳の状態）★
- [ ] 該当ロット隔離、搬送・保管での二次不良に注意

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 機械 | 巻取テンション制御、コンタクトロール圧、芯ズレ |
| 材料 | 幅・厚みばらつき（TS4連動）、表面摩擦 |
| 方法 | テンションテーパ設定、巻取速度 |

**切り分け・確認**
- [ ] 巻取テンション/テーパ設定を確認
- [ ] 幅・厚みばらつきの有無（上流の寸法不良が波及）

**恒久対策の方向**
- [ ] テンション制御の最適化、コンタクト圧・芯合わせ、耳整形
- [ ] 上流寸法安定化（TS4対策）
- [ ] 【要確認】巻き硬さ・ズレの許容基準

関連場面: TS4, D1, TS20

## TS10. 汚れ・油染み・変色
**症状**：油染み、汚れ、黄変・変色、印字/マーキング不良。

**まず確認（初動）**
- [ ] 汚れの種類（油/水/薬品/熱）を特定、位置・頻度を記録 ★
- [ ] 該当ロット隔離、清掃可否/選別可否を品証と判断

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 機械 | 潤滑油・作動油の漏れ・飛散、ロール汚れ |
| 材料 | 原料の変質、添加剤ブリード、酸化 |
| 方法/環境 | 過加熱による黄変、保管環境、手汚れ |

**切り分け・確認**
- [ ] 油染みは漏れ箇所の特定（設備点検）★
- [ ] 変色は熱履歴・材料酸化の切り分け

**恒久対策の方向**
- [ ] 漏れ対策・ロール清掃基準、加熱条件見直し、保管環境改善
- [ ] 【要確認】汚れ・変色の許容基準と判定

関連場面: TS1, D1, D4

## TS11. 臭気（におい）
**症状**：製品からの異臭（樹脂臭・薬品臭・焦げ臭・カビ臭）。用途によりクレーム化。

**まず確認（初動）**
- [ ] においの種類・強さを記録（可能なら複数人で確認）★
- [ ] 該当ロット隔離、用途上の重大性を品証と判断

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 材料 | 樹脂・添加剤・バインダ由来、原料ロット差 |
| 機械/方法 | 過加熱による分解・焦げ、乾燥不足 |
| 環境 | 保管中の吸着・カビ、梱包材のにおい移り |

**切り分け・確認**
- [ ] 材料由来か工程（加熱）由来かを切り分け ★
- [ ] 乾燥・換気・保管条件を確認

**恒久対策の方向**
- [ ] 加熱・乾燥条件の見直し、材料の見直し、保管/梱包改善
- [ ] 【要確認】臭気の評価方法・合否基準（官能／機器）

関連場面: TS10, D4, C1

## TS12. 強度・引張・伸度不足
**症状**：引張強度・伸度・破断強力が規格を下回る。方向（MD/CD）で差。

**まず確認（初動）**
- [ ] 試験方法・試験片・治具・速度が基準どおりか確認（測定起因を除外）★
- [ ] 規格外ロット隔離

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 材料 | 繊維強度・目付・接着のばらつき |
| 機械/方法 | 接着不足（TS8）、目付不足（TS2）、延伸/配向条件 |
| 測定 | 試験機校正・治具・つかみ |

**切り分け・確認**
- [ ] 目付・接着強度と相関を確認（強度不足の主因の多くは目付/接着）★
- [ ] MD/CDの差から配向・条件を推定
- [ ] 試験機の校正・試験条件を確認

**恒久対策の方向**
- [ ] 目付・接着の安定化（TS2/TS8対策）、条件最適化
- [ ] 工程能力（Cp/Cpk）で強度を監視
- [ ] 【要確認】強度規格・試験方法（JIS/顧客規格・試験片・速度）

関連場面: TS2, TS8, D6

## TS13. ピンホール・地合い透け（開孔・薄部）
**症状**：光にかざすと点状の穴（ピンホール）や薄い部分が見える。被覆性・遮蔽性・強度に影響。

**まず確認（初動）**
- [ ] 発生位置の規則性（周期・幅方向の定位置か）を記録★
- [ ] 該当ロット・巻を隔離し、流出範囲を特定

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 材料 | 繊維の分散不良・ダマ、目付不足（TS2） |
| 機械 | ノズル/ダイの詰まり・欠け、ロール傷、エア圧の乱れ |
| 方法 | 地合い条件（TS3）、張力過多での薄伸び |

**切り分け・確認**
- [ ] 目付・地合いと相関（薄部＝局所的な目付低下のことが多い）★
- [ ] 周期性から設備部位（ロール周・ノズル位置）を推定
- [ ] 原料ロット切替との時間相関を確認

**恒久対策の方向**
- [ ] 分散・ろ過の見直し、ノズル/ダイ清掃・点検の標準化
- [ ] 地合い・目付の安定化（TS2/TS3対策）と検査での透過チェック
- [ ] 【要確認】ピンホール・被覆性の社内許容基準・検査方法（透過光・画像）

関連場面: TS2, TS3, D1

## TS14. 静電気・帯電（作業性・異物付着）
**症状**：搬送中に貼り付く・広がる、異物やホコリを引き寄せる、放電でスパーク。乾燥期に増えやすい。

**まず確認（初動）**
- [ ] 帯電の有無・発生工程（搬送/巻取/加工）を確認★
- [ ] 異物付着（TS1）が帯電由来かを切り分け

**起こりやすい要因（4M）**
| 区分 | よくある要因 |
|---|---|
| 環境 | 低湿度（乾燥期）、除電器の能力不足・故障 |
| 機械 | 搬送速度・ロール材質・接触摩擦 |
| 材料 | 樹脂の帯電特性、帯電防止剤の有無 |

**切り分け・確認**
